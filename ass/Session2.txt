Q3.Identify and label the header, main function, and return statement in your hello.c program using comments.

**Code**
// 1. Header file inclusion
#include <stdio.h>

// 2. Main function where program execution begins
int main() {
    
    // Prints the message to the console
    printf("Hello, Instagram World!\n");

    // 3. Return statement indicating successful execution
    return 0;
}
-->Header (#include <stdio.h>): This tells the compiler to include the Standard Input Output library, which is necessary to use the printf function.

-->Main Function (int main()): This is the mandatory entry point of any C program. The code inside the curly braces {} is executed when the program runs.

-->Return Statement (return 0;): This ends the main function and returns a value of 0 to the operating system, signaling that the program executed successfully without any errors.

Q4.Compile your hello.c file using the GCC command line (gcc hello.c -o hello) and run the output file to display the message in your terminal.<br><br><em><strong>Hint:</strong> On Windows, the output file will be hello.exe; on Linux/Mac, it will be hello.</em>

1. Open your Terminal or Command Prompt
Navigate to the directory/folder where you saved your hello.c file using the cd command. For example:
--cd path/to/your/folder

2. Compile the Program
Type the compilation command into your terminal and press Enter:
--gcc hello.c -o hello

3. Run the Output File
Depending on your operating system, type the appropriate command to run the executable:

On Windows:
--hello.exe

On Linux / macOS:
--./hello

Once executed, your terminal will display the output message:
Hello, Instagram World!
